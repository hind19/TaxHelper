Данная утилита (далее по тексту утилита или программа) предназначена для расчета надогообложения ФЛП 3 группы, Украина. Можно ли использовать ее для другисх стран?  Если у Вас налог считается от оборота и есть API чтобы взять официальный курс  - почему нет?
Позволяет обрабатывать валютные поступления в некоторых основных валютах, таких как USD, EUR, PLN, UAH.
Программа имеет два способа ввода данных: ручной и парсинг .csv файла. В последнем случае Вам нужно скачать выписку за нужный период из Вашего банковского приложения.
Утилита сама  получает официальный курс  (НБУ по дефолту) и на основании него производит расчет.
!!! Программа не вносит изменени в какие - либо данные. !!!!

Программа имеет некотоую возможность настроек.
Для доступа к ним Вам нужно отредактировать файл TaxHelper.dll.config  в выходной диектории программы.

Перечень настроек:
BankCurrenceExchangeUrl -  ссылка на АПИ центрального банка.
CurrencyCodeParam и  DateParam -  названия параметров командной строки для запроса курса валюты. Соответственно кода валюты и даты на которую нужно взять курс
TaxRate и MilitaryTaxRate - ставки единого налога и военнного сбора.

PaymentDateColumn PaymentAmountColumn PaymentCurrencyColumn - названия колонок .csv файла, по которым выподлгняется расчет. В каждом банке колонки будут иметь свои названия.
ColumnSplitter - сплиттер данных .csv файла. Оказывается, некоторые особо одаренные банкиры  не желают использовать стандартную ',' как сплиттер.
